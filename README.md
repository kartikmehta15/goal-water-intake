A beautiful and interactive website to track your daily water consumption and stay hydrated!

## Features

- **Daily Water Logging**: Easily log your water intake for any day with a customizable daily goal
- **Visual Progress Tracking**: See your hydration progress with an animated creature that changes daily
- **Interactive Calendar View**: Navigate through months and view your hydration history at a glance
- **Color-Coded Achievement Levels**: Days are color-coded based on goal achievement (100%+, 75-99%, 50-74%, 25-49%, <25%)
- **Summary Statistics**: Track your overall performance with comprehensive statistics
- **Edit Historical Data**: Click any day in the calendar to edit or delete past entries
- **CSV Export**: Download your complete water intake data as a CSV file
- **Persistent Storage**: All data is stored locally in your browser using localStorage
- **31 Unique Creatures**: Each day features a different animal, plant, or creature icon that fills proportionally to your goal

## How to Use

1. Open `index.html` in your web browser
2. Enter the date, amount of water consumed (in ml), and your daily goal
3. Click "Save Entry" to log your water intake
4. View your progress in the "Today's Progress" section
5. Check the calendar view to see your hydration history
6. Click any day in the calendar to edit or view past entries
7. Export your data using the "Export to CSV" button

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- LocalStorage API for data persistence

## Getting Started

Simply open the `index.html` file in any modern web browser. No installation or server required!
